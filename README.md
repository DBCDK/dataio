dataIO
======

### Development

**Requirements**

To build this project, JDK 21 and Apache Maven are required.

**_NOTE:_**  Special job-store-service testing setup. see more in [job-store-service/README.md](job-store-service/README.md)

**Scripts**
* clean - clears build artifacts
* build - builds and tests artifacts (including docker images)

```bash
./clean && ./build
```
